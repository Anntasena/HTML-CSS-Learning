(( COMPONENT AND LAYOUT RULES ))

( Dari Elements ke Webpage )

Ketika kita ingin memulai situs web baru, dan mulai dengan halaman kosong, dan yang kita miliki hanyalah konten, kita mungkin akan merasa tersesat. Tetapi kita tidak perlu merasa tersesat, tenang saja kita tidak perlu merasa sesast, karna disinilah pola-pola (pattern) yang sudah stabil berperan. Jadi pada dasarnya kita membangun situs web selalu dengan cara yang sama

Kita bisa memulai menbangun website:

> 1. Mulai dari elemnent kecil
      - Paragraf
      - Images
      - Button
> 2. Kemudian susun element-element kedalam component umum
      - Feature
      - Card
      - Pricing Table
      - component lainnya
> 3. Lalu kita mengambil semua component yang kita rancang, dan menggunaaknnya untuk membangun layout
> 4. Gabungkan berberapa layout dan atur sesuai panduan untuk menjadi webpage

KESIMPULAN
----------
element-element =(diorganisasikan)=> component =(diorganisasikan)=> layout =(digabungkan)=> webpage
